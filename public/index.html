What it does
🌐
Full Web Proxy

Fetches any public URL server-side, rewrites all links and assets so you can browse normally through the proxy.

🔗
Link Rewriting

Every href, src, action, srcset, and CSS url() is rewritten so navigation stays inside the proxy automatically.

🔒
Private IPs Blocked

Requests to localhost, 192.168.x.x, 10.x.x.x, and other internal addresses are rejected for safety.

⚡
Fast & Lightweight

Pure Node.js with zero heavyweight dependencies. Streams binary resources directly without unnecessary buffering.

🚦
Rate Limited

120 requests per minute per IP. Keeps the server healthy and prevents abuse.

Live Metrics
